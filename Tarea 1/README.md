# Tarea 1  Diagnostico experimental de paralelismo

## Requisitos
- Conda instalado

1. Crear entorno:
   conda env create -f environment.yml
2. Activar entorno:
   conda activate lab1_paralela
3. Abrir Jupyter:
   jupyter notebook
4. Abrir el archivo Tarea_1.ipynb
5. Ejecutar las celdas en orden, de arriba hacia abajo

- Cada ejercicio usa seed fija
